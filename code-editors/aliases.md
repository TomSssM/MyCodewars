# Aliases

Here is a list of my most commonly used aliases:

__.bashrc:__

git aliases:

```bash
alias st='git status'
alias com='git add -A && git commit -m'
alias amm='git add -A && git commit --amend'
alias br='git branch -m'
alias pl='git pull'
alias p='git push'
alias c='git checkout'
alias l='git log'
alias ch='git checkout ./'
```

npm aliases:
```
alias bld='npm run build'
alias st="npm start"
alias ts="npm run test"
```

__.gitconfig:__

```bash
[alias]
  st = status
  cm = commit
  a = add -A
  cma = commit --amend
  p = push
  l = log
  c = checkout
  b = branch
  ba = branch -a
  n = branch -m
  ch = checkout ./
```
